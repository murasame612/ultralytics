## 基于yolov8-obb的目标检测和分割的模型训练

**源码来自ultralytics的开源项目YOLOv8**

首先我们需要准备好具有标注好目标框的标注文件格式为txt，

标注文件的标准格式为：

| class_id | x_center   | y_center   | width      | height     |
|----------|------------|------------|------------|------------|
| -        | 要求为0-1的归一化 | 要求为0-1的归一化 | 要求为0-1的归一化 | 要求为0-1的归一化 |
